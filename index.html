Setup Guide
1

Terima Email Campaign

Simpan file Excel attachment dari email (berisi daftar toko alokasi). Salin juga link Google Sheet dari email.

2

Tambah Campaign di ERA-VIS

Klik Campaigns → + Tambah Campaign. Pilih mode "Upload Excel", drag & drop file Excel dari email, lalu paste link Google Sheet. Selesai.

3

Publish Google Spreadsheet

Agar ERA-VIS bisa baca status submit: buka Google Sheet → File → Share → Publish to web → CSV → Publish.

4

Setup Fonnte (WhatsApp Reminder)

Settings → masukkan API Token dari fonnte.com untuk aktifkan Auto Reminder.

⚡ Flow Otomatis — Tanpa Buat Google Sheet Baru

ERA-VIS membaca daftar toko langsung dari file Excel attachment email. Google Sheet hanya dipakai untuk membaca response form. Setiap campaign baru = upload Excel + paste 1 link.
